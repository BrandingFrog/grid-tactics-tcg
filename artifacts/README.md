# Local artifacts

This folder is the local home for browser screenshots, temporary exports, and investigation notes that do not belong in Git history.

- `browser-captures/` contains screenshots previously created at repository root.
- `notes/` contains temporary setup or visual-review notes.
- `debug/` contains disposable intermediate data.

Everything below this README is ignored by Git. Durable evidence that supports a tracked plan belongs under `.planning/debug/` instead.
