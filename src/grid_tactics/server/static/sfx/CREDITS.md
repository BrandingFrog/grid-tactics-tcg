# Sound Effects

All audio files in this directory are sourced from [Kenney.nl](https://kenney.nl/) and released under [Creative Commons CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/) (public domain -- no attribution required).

Warm re-source 2026-07-10 (user: "the sounds are too chirpy"): the original
Interface-Sounds beeps were replaced with physical card-table / cloth /
wood sounds matching the candlelit-parchment theme. Every file is
loudness-normalized to -18 LUFS (ffmpeg `loudnorm`) so the mixed packs sit
at one level. Victory/defeat were picked from the Pizzicato jingles by
pitch-trajectory analysis (victory ascends ~711->1347 Hz, defeat descends
~1256->582 Hz).
