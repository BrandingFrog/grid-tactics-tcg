  <name>Task 1: Create sync_cards.py with pure-function wikitext generation</name>
  <files>wiki/sync/sync_cards.py</files>
  <action>
Create `wiki/sync/sync_cards.py` with the following functions:

1. **`build_card_name_map(cards_dir: Path) -> dict[str, str]`**
   - Reads all `data/cards/*.json` files
   - Returns `{card_id: display_name}` mapping (e.g., `{"rat": "Rat", "ratchanter": "Ratchanter"}`)
   - Used for cross-link resolution

2. **`derive_keywords(card: dict) -> list[str]`**
   - Derives keywords from card JSON structure (never hard-coded per card)
   - Mapping rules (from research):
     - `card_type == "react"` or `react_condition` present -> "React"
     - `unique == true` -> "Unique"
     - `range > 0` (minion) -> "Ranged" ; `range == 0` (minion) -> "Melee"
     - `tutor_target` present -> "Tutor"
     - `transform_options` present -> "Transform"
     - `promote_target` present -> "Promote"
     - `activated_ability` present -> "Active"
     - `effects` with `trigger: "on_play"` AND card_type == "minion" -> "Summon"
     - `effects` with `trigger: "on_death"` -> "Death"
     - `effects` with `trigger: "passive"` -> "Passive"
     - `activated_ability.effect_type` starts with "conjure" -> "Conjure"
     - `summon_sacrifice_tribe` present -> "Sacrifice"
     - Effect types: `"burn"` -> "Burn", `"heal"` -> "Heal", `"damage"` -> "Deal", `"destroy"` -> "Destroy", `"negate"` -> "Negate", `"leap"` -> "Leap", `"rally_forward"` -> "Rally", `"deploy_self"` -> "Deploy", `"grant_dark_matter"` -> "Dark Matter"
   - Return sorted, deduplicated list
   - Do NOT add keywords for magic/react cards based on range (they don't have range stats)

3. **`build_rules_text(card: dict, name_map: dict[str, str] | None = None) -> str`**
   - Synthesizes human-readable rules text from card JSON fields
   - Uses an `EFFECT_TEMPLATES` dict mapping effect types to sentence patterns:
     - `"damage"`: "Deal {amount} damage to a target."
     - `"heal"`: "Restore {amount} HP."
     - `"burn"`: "Apply {amount} burn damage over time."
     - `"negate"`: "Negate the target spell."
     - `"destroy"`: "Destroy a target minion."
     - `"leap"`: "Leap {amount} spaces forward."
     - `"rally_forward"`: "Push all friendly minions forward {amount} space(s)."
     - `"deploy_self"`: "Deploy this card to the board."
     - `"grant_dark_matter"`: "Grant Dark Matter to a target."
     - `"buff_health"`: "Grant +{amount} HP to a target."
     - `"dark_matter_buff"`: "Buff a minion with Dark Matter (+{amount})."
     - `"passive_heal"`: "Passively heal for {amount} each turn."
     - `"promote"`: "On death: promote to {target}." (use wikilink if name_map provided)
     - `"tutor"`: "On play: search your deck for {target} and add it to your hand." (wikilink)
   - For `activated_ability`: "'''Active:''' Pay {cost} mana to {ability_name}." If `summon_card_id` present, append wikilink to summoned card.
   - For `transform_options`: "'''Transform:''' {list of 'Pay X mana -> [[Card:Name|Name]]' joined by ', '}."
   - For `react_condition`: Prefix with react condition text (e.g., "When opponent plays a magic card: ")
   - Combine multiple effects with space separation
   - Cross-link targets use `[[Card:Display Name|Display Name]]` format via `name_map`

4. **`card_to_wikitext(card: dict, name_map: dict[str, str] | None = None, art_exists: bool = True) -> str`**
   - Builds complete `{{Card ... }}` template invocation
   - Field mapping (from research JSON Field -> Template Param table):
     - `card["name"]` -> `name`
     - `card["card_type"]` capitalized -> `type`
     - `card["element"]` capitalized -> `element`
     - `card.get("tribe", "")` -> `tribe`
     - `card.get("mana_cost", "")` -> `cost`
     - `card.get("attack", "")` -> `attack` (omit for magic/react)
     - `card.get("health", "")` -> `hp` (omit for magic/react)
     - `card.get("range", "")` -> `range` (omit for magic/react — check card_type, don't just check if field exists since range=0 is valid for minions)
     - `build_rules_text(card, name_map)` -> `rules`
     - `card.get("flavour_text", "")` -> `flavor` (note British->American spelling)
     - `", ".join(derive_keywords(card))` -> `keywords`
     - `f"{card['card_id']}.png"` if `art_exists` else omit (lets template fall back to CardBack.png)
     - Read version from `VERSION.json` or default "0.4.2" -> `patch`
     - `card.get("card_id", "")` -> `stable_id`
     - `"true"` if `card.get("deckable", True)` else `"false"` -> `deckable`
   - Only emit fields with non-empty values (skip empty strings, None)
   - Format: `| {key:9s}= {value}` per line, wrapped in `{{Card ... }}`
   - Do NOT append `[[Category:Cards]]` or Notes section — Template:Card handles categories

5. **`get_version() -> str`**
   - Read `src/grid_tactics/server/static/VERSION.json` and return the `version` field
   - Fall back to `"0.4.2"` if file not found

Important: Do NOT import from `src/grid_tactics/` card loader. Read JSON directly with `json.load()`. The card loader uses IntEnums; wiki sync needs raw string values.
  </action>
