   ```
   {{Card
   | name     = Ratchanter
   | type     = Minion
   | element  = Dark
   | tribe    = Rat
   | cost     = 3
   | attack   = 2
   | hp       = 4
   | range    = 1
   | rules    = Whenever a friendly Rat dies, draw a card.
   | flavor   = The rats listen, and the rats remember.
   | keywords = Sacrifice, Summon
   | patch    = 0.4.2
   | stable_id= ratchanter
   }}
   
   == Notes ==
   This page is a Phase 1 sample created by wiki/sync/create_sample_card.py.
   It is NOT generated from data/cards/ — Phase 3 will replace it.
   
   [[Category:Cards]]
   ```

3. After edit, runs a verification `site.ask("[[CardType::Minion]][[Element::Dark]]|?Cost|?HP|limit=10")` and asserts Ratchanter is in the results with Cost=3, HP=4 — prints PASS/FAIL

5. Prints the URL of the created page for the human checkpoint

**Important:** Do NOT include `#subobject` on this sample (no effects on Ratchanter in this simplified sample). Effects/subobjects get exercised in Phase 3 when sync_cards.py hits real data. Phase 1's job is to prove the template + basic properties work.

Run: `cd wiki && python -m sync.create_sample_card`. Confirm PASS output and note the URL.
