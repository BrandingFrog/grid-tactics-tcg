**Rules/conceptual page generator:**

Add `RULES_PAGES: dict[str, str]` constant containing the 6 conceptual pages with real content. Each page should have substantive content (not stubs), drawing from the game rules in CLAUDE.md. The pages are:

1. **"Grid Tactics TCG"** (overview): Brief overview of the game -- fantasy TCG on a 5x5 grid, two players, deploy minions, cast magic, react cards. Mention the 7 elements and card types (Minion, Magic, React).

2. **"5x5 Board"**: Explain the 5-column, 5-row grid. Each player controls the row closest to them. Minions move forward in their lane (column). Columns are lanes. Position matters for attacks and sacrifices.

3. **"Mana"**: Mana is the resource for playing cards. Players gain mana each turn. Mana banking -- unspent mana carries over. One shared mana pool, not per-element.

4. **"React Window"**: After each action, the opponent gets a react window to play react cards. React cards can counter spells, deploy minions, or trigger effects. If no react card is played, the window closes and play continues.

5. **"Win Conditions"**: Two ways to win -- sacrifice (move a minion across the entire board to the opponent's back row) or HP depletion (reduce opponent's HP to 0 through attacks and effects).

6. **"Turn Structure"**: Auto-draw at turn start, then one action per turn (play a card, move a minion, attack, sacrifice, or pass). React window after each action.

Each page MUST end with:
```
{{ManuallyMaintained}}
[[Category:Rules]]
```

Add `sync_rules_pages(site, dry_run: bool = False) -> dict`: Upsert the 6 rules pages.
