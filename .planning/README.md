# Planning workspace

This folder contains the GSD project record:

- `ROADMAP.md`, `STATE.md`, `PROJECT.md`, and `REQUIREMENTS.md` describe current direction and status.
- `phases/` contains scoped phase research, plans, summaries, and verification.
- `quick/` contains smaller one-off plans.
- `debug/` contains investigations and durable test evidence.
- `research/` contains reusable technical research.

Keep transient browser captures under `artifacts/`; place only evidence that supports a tracked decision or verification here.
