<objective>
Extend react_condition matching so react cards fire only when their written condition matches the CURRENT react window's trigger. Add three new ReactCondition values (OPPONENT_SUMMONS_MINION, OPPONENT_START_OF_TURN, OPPONENT_END_OF_TURN) to enable more expressive card designs in future iterations.

Purpose: Today _check_react_condition matches against `state.pending_action.action_type`. That works for ACTION-phase actions but doesn't distinguish between a MAGIC cast and a MINION summon declaration (both push originators now). After 14.7-02 added react_context, conditions can be checked against the CURRENT context. This plan wires the condition matching to react_context AND adds the three new enum values for future card expressivity.

Step 3 — Read src/grid_tactics/legal_actions.py. Find _check_react_condition (per RESEARCH §1 line 654-747). The existing function takes a react condition + state and checks against state.pending_action.action_type. Extend it to use state.react_context:

```python
def _check_react_condition(
    condition: ReactCondition, state: GameState, card_def: CardDefinition, library: CardLibrary,
) -> bool:
    """Return True if this card's react condition matches the current react window.

    Uses state.react_context (14.7-02) as the primary signal for WHAT kind of
    react window is currently open. Falls back to state.pending_action for
    backward compat (pre-14.7 conditions).
    """
    from grid_tactics.enums import ReactContext

    # Existing ANY_ACTION short-circuit
    if condition == ReactCondition.ANY_ACTION:
        return True

    # 14.7 new conditions — dispatch on react_context
    if condition == ReactCondition.OPPONENT_SUMMONS_MINION:
        return state.react_context == ReactContext.AFTER_SUMMON_DECLARATION

    if condition == ReactCondition.OPPONENT_START_OF_TURN:
        return state.react_context == ReactContext.AFTER_START_TRIGGER

    if condition == ReactCondition.OPPONENT_END_OF_TURN:
        return state.react_context == ReactContext.BEFORE_END_OF_TURN

    # Existing conditions — dispatch on react_context where it clarifies,
    # fall back to pending_action otherwise.
    if condition == ReactCondition.OPPONENT_PLAYS_MAGIC:
        # Magic cast pushes an originator; the react_stack has an is_originator entry
        # with origin_kind == "magic_cast". Check the originator in state.react_stack.
        if state.react_context == ReactContext.AFTER_ACTION:
            for entry in state.react_stack:
                if getattr(entry, "is_originator", False) and getattr(entry, "origin_kind", None) == "magic_cast":
                    return True
        return False

    if condition == ReactCondition.OPPONENT_PLAYS_MINION:
        # Backward-compat: matches during AFTER_SUMMON_DECLARATION
        # (which is where minion deployment opens a window).
        return state.react_context == ReactContext.AFTER_SUMMON_DECLARATION

    # Element-based conditions (OPPONENT_PLAYS_WOOD etc.):
    # Check against originator's card element if in AFTER_ACTION / AFTER_SUMMON_DECLARATION
    element_map = {
        ReactCondition.OPPONENT_PLAYS_WOOD: Element.WOOD,
        ReactCondition.OPPONENT_PLAYS_FIRE: Element.FIRE,
        ReactCondition.OPPONENT_PLAYS_EARTH: Element.EARTH,
        ReactCondition.OPPONENT_PLAYS_WATER: Element.WATER,
        ReactCondition.OPPONENT_PLAYS_METAL: Element.METAL,
        ReactCondition.OPPONENT_PLAYS_DARK: Element.DARK,
        ReactCondition.OPPONENT_PLAYS_LIGHT: Element.LIGHT,
    }
    if condition in element_map:
        target_element = element_map[condition]
        # Find the originator (magic or summon) and check its card's element
        for entry in state.react_stack:
            if getattr(entry, "is_originator", False):
                card_def = library.get_by_id(entry.card_numeric_id)
                if card_def.element == target_element:
                    return True
        return False

    # OPPONENT_ATTACKS / OPPONENT_SACRIFICES / OPPONENT_DISCARDS / OPPONENT_ENDS_TURN
    # use state.pending_action.action_type as before (these don't push originators)
    if state.pending_action is None:
        return False
    pa_type = state.pending_action.action_type

    if condition == ReactCondition.OPPONENT_ATTACKS:
        return pa_type == ActionType.ATTACK
    if condition == ReactCondition.OPPONENT_SACRIFICES:
        return pa_type == ActionType.SACRIFICE
    if condition == ReactCondition.OPPONENT_DISCARDS:
        return pa_type == ActionType.DRAW  # Or whatever indicates a discard
    if condition == ReactCondition.OPPONENT_ENDS_TURN:
        return pa_type == ActionType.PASS
    if condition == ReactCondition.OPPONENT_PLAYS_REACT:
        # Last entry on stack is a non-originator REACT card → another react can be played
        if len(state.react_stack) > 0:
            last = state.react_stack[-1]
            return not getattr(last, "is_originator", False)
        return False

    return False
```
