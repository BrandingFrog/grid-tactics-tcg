  ```python
  import copy

  def filter_state_for_spectator(state_dict: dict, god_mode: bool, perspective_idx: int = 0) -> dict:
      """Filter game state for a spectator.

      god_mode=True: return a deep copy of full state with no filtering.
        Both hands, both decks (with card order), pending tutor matches all visible.
        Adds top-level flag 'spectator_god_mode': True for client UI.

      god_mode=False: return filter_state_for_player(state, perspective_idx).
        Spectator sees what player perspective_idx sees (default 0 = Player 1).
        Adds 'spectator_god_mode': False.

      Always adds top-level 'is_spectator': True so the client knows to enter spectator mode.
      """
      if god_mode:
          filtered = copy.deepcopy(state_dict)
          # Ensure pending tutor matches stay as full lists (mirror what filter_state_for_player
          # would strip for the opponent). They are already present in state_dict; deepcopy keeps them.
      else:
          filtered = filter_state_for_player(state_dict, perspective_idx)
      filtered["is_spectator"] = True
      filtered["spectator_god_mode"] = god_mode
      filtered["spectator_perspective"] = perspective_idx
      return filtered
  ```

  Verify the existing filter_state_for_player handles tutor pending correctly (read lines around it). If it strips tutor matches for the non-acting player, the non-god spectator path inherits that behavior automatically — desired.
