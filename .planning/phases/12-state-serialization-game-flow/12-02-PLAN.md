    ```python
    def _build_card_defs(library):
        defs = {}
        for nid in range(len(library)):
            try:
                card = library.get_by_id(nid)
                defs[nid] = {
                    "name": card.name,
                    "card_type": int(card.card_type),
                    "mana_cost": card.mana_cost,
                    "attack": card.attack,
                    "health": card.health,
                    "attack_range": card.attack_range,
                    "element": int(card.element) if card.element is not None else None,
                }
            except (KeyError, IndexError):
                break
        return defs
    ```

    `_emit_state_to_players(session, state)` -- per D-07, emits filtered state + legal actions to each player via their SID:
    ```python
    def _emit_state_to_players(session, state):
        state_dict = state.to_dict()
        actions = legal_actions(state, session.library) if not state.is_game_over else ()
        serialized_actions = [serialize_action(a) for a in actions]

        # Determine decision-maker: REACT phase uses react_player_idx, ACTION uses active_player_idx
        if state.phase == TurnPhase.REACT:
            decision_idx = state.react_player_idx
        else:
            decision_idx = state.active_player_idx

        for idx in (0, 1):
            filtered = filter_state_for_player(state_dict, idx)
            emit("state_update", {
                "state": filtered,
                "legal_actions": serialized_actions if idx == decision_idx else [],
                "your_player_idx": idx,
            }, to=session.player_sids[idx])
    ```

    `_emit_game_over(session, state)` -- emits game_over event per D-02 (same filter):
    ```python
    def _emit_game_over(session, state):
        state_dict = state.to_dict()
        for idx in (0, 1):
            filtered = filter_state_for_player(state_dict, idx)
            emit("game_over", {
                "winner": int(state.winner) if state.winner is not None else None,
                "final_state": filtered,
                "your_player_idx": idx,
            }, to=session.player_sids[idx])
    ```

    **3. Add submit_action handler (VIEW-02, VIEW-03, SERVER-03):**
    Register `@socketio.on("submit_action")` inside `register_events()`. Per RESEARCH.md complete handler pattern:

    a. Look up token from SID via `_room_manager.get_token_by_sid(request.sid)`
    b. Look up room_code via `_room_manager.get_room_code_by_token(token)`
    c. Get game session via `_room_manager.get_game(room_code)`
    d. Get player_idx via `session.get_player_idx(token)`
    e. Check `state.is_game_over` -- if true, emit error "Game is already over"
    f. Determine decision_idx: if phase==REACT use react_player_idx, else active_player_idx
    g. If player_idx != decision_idx, emit error "Not your turn"
    h. Try `reconstruct_action(data)` -- on ValueError/KeyError/TypeError emit error
    i. Acquire `session.lock`:
       - Compute `legal_actions(session.state, session.library)`
       - Check action membership (action in valid_actions) -- if not, emit error "Illegal action", return
       - Apply: `session.state = resolve_action(session.state, action, session.library)`
       - Auto-pass loop: while not game_over and legal_actions returns empty, auto-submit pass_action()
       - Capture new_state = session.state
    j. Release lock, then call `_emit_state_to_players(session, new_state)`
    k. If `new_state.is_game_over`, call `_emit_game_over(session, new_state)`
