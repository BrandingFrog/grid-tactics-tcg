       **`handle_react_action(state: GameState, action: Action, library: CardLibrary) -> GameState`:**
       - If `action.action_type == ActionType.PASS`:
         - Call `resolve_react_stack(state, library)` to resolve LIFO and advance turn.
       - If `action.action_type == ActionType.PLAY_REACT`:
         - Validate `state.phase == TurnPhase.REACT`.
         - Get reacting player: `state.players[state.react_player_idx]`.
         - Get card: `card_numeric_id = player.hand[action.card_index]`, look up card_def.
         - Validate card is react-eligible:
           - `card_def.card_type == CardType.REACT`: use `card_def.mana_cost` and `card_def.effects`.
           - `card_def.is_multi_purpose`: use `card_def.react_mana_cost` and `card_def.react_effect`.
           - Otherwise raise ValueError.
         - Validate sufficient mana (use appropriate cost).
         - Validate stack depth < MAX_REACT_STACK_DEPTH (raise ValueError if at cap, per research Pitfall 3).
         - Spend mana on reacting player, discard card from hand to graveyard.
         - Create ReactEntry, append to react_stack tuple.
         - Switch react_player_idx to `1 - state.react_player_idx` (counter-react opportunity per D-05).
         - Return updated state.
       - Any other action_type during REACT raises ValueError.

       **`resolve_react_stack(state: GameState, library: CardLibrary) -> GameState`:**
       - Iterate over `reversed(state.react_stack)` (LIFO per D-06).

       - After resolving all entries, cleanup dead minions (reuse `_cleanup_dead_minions` from action_resolver).

    1. **Create src/grid_tactics/legal_actions.py** with the main function and helpers:

       **`legal_actions(state: GameState, library: CardLibrary) -> tuple[Action, ...]`:**
       - If `state.phase == TurnPhase.ACTION`: call `_action_phase_actions(state, library)`.
       - If `state.phase == TurnPhase.REACT`: call `_react_phase_actions(state, library)`.

       **`_action_phase_actions(state, library) -> tuple[Action, ...]`:**
       - `actions: list[Action] = []`
       - Get active player: `state.active_player`.

       **PLAY_CARD enumeration:**
       - For each `(idx, card_numeric_id)` in `enumerate(player.hand)`:
         - `card_def = library.get_by_id(card_numeric_id)`
         - If `player.current_mana < card_def.mana_cost`: skip (D-11).
         - If `card_def.card_type == CardType.MINION`:
           - Get valid deploy positions via `_valid_deploy_positions(state, card_def, player.side)`:
             - If `card_def.attack_range == 0` (melee per D-08): all empty cells in `Board.get_positions_for_side(side)` (friendly rows 0-1 for P1, 3-4 for P2).
             - If `card_def.attack_range >= 1` (ranged per D-09): only empty cells in back row (row `BACK_ROW_P1` for P1, `BACK_ROW_P2` for P2).
           - For each valid position:
             - Check if card has ON_PLAY effects with SINGLE_TARGET: if so, enumerate target_pos for each enemy minion on board.
             - Otherwise: `actions.append(play_card_action(card_index=idx, position=pos))`.
         - If `card_def.card_type == CardType.MAGIC`:
           - Check if any effect has `target == TargetType.SINGLE_TARGET`:
             - Yes: enumerate enemy minion positions as target_pos. For each: `actions.append(play_card_action(card_index=idx, target_pos=target_pos))`.
             - No: `actions.append(play_card_action(card_index=idx))`.
         - Skip `CardType.REACT` during ACTION phase.

       **MOVE enumeration:**
       - For each minion in `state.get_minions_for_side(player.side)`:
         - For each `adj_pos` in `Board.get_orthogonal_adjacent(minion.position)`:
           - If `state.board.get(adj_pos[0], adj_pos[1]) is None` (empty cell):
             - `actions.append(move_action(minion_id=minion.instance_id, position=adj_pos))`

       **ATTACK enumeration:**
       - For each minion in `state.get_minions_for_side(player.side)`:
         - `card_def = library.get_by_id(minion.card_numeric_id)`
         - For each enemy_minion in state.minions where `enemy.owner != player.side`:

       **DRAW:**
       - If `player.deck` is non-empty: `actions.append(draw_action())`.

       **PASS:**
       - Always: `actions.append(pass_action())` (D-16).

         - `card_def = library.get_by_id(card_numeric_id)`
         - If `card_def.card_type == CardType.REACT` and `player.current_mana >= card_def.mana_cost`:
           - If any effect has SINGLE_TARGET: enumerate enemy minion positions. For each: `actions.append(play_react_action(card_index=idx, target_pos=pos))`.
           - Else: `actions.append(play_react_action(card_index=idx))`.
         - If `card_def.is_multi_purpose` and `player.current_mana >= card_def.react_mana_cost`:
           - If react_effect has SINGLE_TARGET: enumerate targets.
           - Else: `actions.append(play_react_action(card_index=idx))`.
       - Check stack depth < MAX_REACT_STACK_DEPTH; if at cap, only PASS is legal.
       - Always: `actions.append(pass_action())`.
