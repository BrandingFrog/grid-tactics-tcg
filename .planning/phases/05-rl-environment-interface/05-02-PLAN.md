    **src/grid_tactics/rl/env.py** -- per D-07, D-08, D-09:

    Create `GridTacticsEnv(gymnasium.Env)` class:

    ```python
    class GridTacticsEnv(gymnasium.Env):
        metadata = {"render_modes": []}
    ```

    **__init__(self, library, deck_p1, deck_p2, seed=42, turn_limit=DEFAULT_TURN_LIMIT)**:
    - Store library, decks, seed, turn_limit
    - Create ActionEncoder instance
    - Define observation_space = gymnasium.spaces.Box(low=-1.0, high=1.0, shape=(OBSERVATION_SIZE,), dtype=np.float32)
    - Define action_space = gymnasium.spaces.Discrete(ACTION_SPACE_SIZE)
    - Initialize self.state = None, self.rng = None

    **reset(self, seed=None, options=None) -> tuple[np.ndarray, dict]**:
    - Call super().reset(seed=seed)
    - Use provided seed if not None, otherwise self._seed
    - self.state, self.rng = GameState.new_game(game_seed, self.deck_p1, self.deck_p2)
    - Build observation for current acting player: encode_observation(self.state, self.library, self._current_player_idx())
    - Build mask via build_action_mask
    - Return (obs, {"action_mask": mask})

    **step(self, action_int: int) -> tuple[np.ndarray, float, bool, bool, dict]**:
    - Determine which player is acting (self._current_player_idx())
    - Decode action: self.action_encoder.decode(action_int, self.state, self.library)
    - Apply: self.state = resolve_action(self.state, action, self.library)
    - Check terminated = self.state.is_game_over
    - Check truncated = self.state.turn_number > self.turn_limit
    - Compute reward for the player who just acted: compute_reward(self.state, acting_player_idx)
    - If not terminated and not truncated: build obs for next acting player, build mask
    - If terminal: build final obs (can use zeros or last state), mask all zeros except PASS
    - Return (obs, reward, terminated, truncated, {"action_mask": mask})

    **action_masks(self) -> np.ndarray**:
    - Return build_action_mask(self.state, self.library, self.action_encoder)

    **_current_player_idx(self) -> int**:
    - If phase == ACTION: return state.active_player_idx
    - If phase == REACT: return state.react_player_idx

    Key design: The environment treats BOTH players as the same agent alternating turns. Observation is always from the perspective of whoever acts next. This follows the PettingZoo Connect Four pattern from the research. The reward is from the perspective of the player who just acted (the one whose action resolved).
