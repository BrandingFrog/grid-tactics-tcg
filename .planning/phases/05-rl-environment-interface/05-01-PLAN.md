    **src/grid_tactics/rl/action_space.py** -- per D-04, D-05, D-06:
    Define section base constants from research:
    ```python
    MAX_HAND_SIZE = 10
    PLAY_CARD_BASE = 0       # 250 slots: hand(10) * grid(25)
    MOVE_BASE = 250          # 100 slots: source(25) * dir(4)
    ATTACK_BASE = 350        # 625 slots: source(25) * target(25)
    SACRIFICE_BASE = 975     # 25 slots: source(25)
    DRAW_IDX = 1000          # 1 slot
    PASS_IDX = 1001          # 1 slot
    REACT_BASE = 1002        # 260 slots: hand(10) * target_or_none(26)
    ACTION_SPACE_SIZE = 1262
    ```

    Helper functions:
    - `pos_to_flat(pos: tuple[int,int]) -> int`: row * GRID_COLS + col
    - `flat_to_pos(flat: int) -> tuple[int,int]`: (flat // GRID_COLS, flat % GRID_COLS)
    - DIRECTION_MAP = {(-1,0): 0, (1,0): 1, (0,-1): 2, (0,1): 3}
    - DIRECTION_REVERSE = {v:k for k,v in DIRECTION_MAP.items()}

    Implement `ActionEncoder` class:
    - `encode(action: Action, state: GameState) -> int`:
      - PASS -> PASS_IDX
      - DRAW -> DRAW_IDX
      - PLAY_CARD -> PLAY_CARD_BASE + hand_idx * GRID_SIZE + cell (where cell is deploy pos for minion, target pos for targeted magic, 0 for untargeted magic)
      - MOVE -> MOVE_BASE + source_flat * 4 + direction (compute direction from minion.position to action.position)
      - ATTACK -> ATTACK_BASE + source_flat * GRID_SIZE + target_flat (look up attacker and defender positions from state)
      - SACRIFICE -> SACRIFICE_BASE + source_flat (look up minion position from state)
      - PLAY_REACT -> REACT_BASE + hand_idx * 26 + (target_flat if targeted, else 25)

    - `decode(action_int: int, state: GameState, library: CardLibrary) -> Action`:
      - Determine section by range comparison
      - For PLAY_CARD: extract hand_idx and cell_flat; look up card type from state.active_player.hand[hand_idx] via library; if MINION -> position=cell_pos; if MAGIC with SINGLE_TARGET -> target_pos=cell_pos; if MAGIC untargeted -> neither
      - For MOVE: extract source_flat and direction; find minion at source position via state; return move_action(minion_id, dest_pos)
      - For ATTACK: extract source_flat and target_flat; find minion at source and target; return attack_action(minion_id, target_id)
      - For SACRIFICE: extract source_flat; find minion at position; return sacrifice_action(minion_id)
      - For PLAY_REACT: extract hand_idx and target; if target < 25 -> target_pos=flat_to_pos(target); else None

    Implement `build_action_mask(state: GameState, library: CardLibrary, encoder: ActionEncoder) -> np.ndarray`:
    - mask = np.zeros(ACTION_SPACE_SIZE, dtype=np.bool_)
    - Call legal_actions(state, library) to get all valid actions
    - For each action, encode to int via encoder.encode(action, state) and set mask[idx] = True
    - Return mask

    - Not game over -> 0.0
    - Draw (winner is None) -> 0.0
    - Player won -> +1.0
    - Player lost -> -1.0
