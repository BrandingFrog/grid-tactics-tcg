    Extend `src/grid_tactics/rl/reward.py` (do NOT replace existing compute_reward -- add alongside it):

    1. Add `potential(state: GameState, player_idx: int) -> float` function with 4 components (per D-04, D-06):
       - **HP advantage** (weight 0.3): `(my_hp - opp_hp) / (STARTING_HP * 2)` -- normalized HP difference
       - **Board control** (weight 0.3): `(my_minions - opp_minions) / 10.0` -- minion count advantage
       - **Mana efficiency** (weight 0.2): `my_current_mana / MAX_MANA_CAP` -- having mana is good
       - **Positional advancement** (weight 0.2): `avg_row_progress / 4.0` -- minions closer to opponent back row
         - Player 0 advancing = row increasing (toward row 4)
         - Player 1 advancing = row decreasing (toward row 0)
         - Divide by max(my_minions, 1) to average
       - Return weighted sum, clamped to [-1.0, 1.0]
       - Per Pitfall 3: Total output must be small relative to terminal +1/-1

    2. Add `compute_shaped_reward(prev_state: GameState, new_state: GameState, player_idx: int, gamma: float = 0.99) -> float`:
       - `base = compute_reward(new_state, player_idx)`
       - `shaping = gamma * potential(new_state, player_idx) - potential(prev_state, player_idx)`
       - Return `base + shaping`
       - Per D-06: This is the canonical potential-based formulation from Ng et al., 1999

    3. Write tests FIRST in `tests/test_reward_shaping.py`:
       - Use `make_game_state_with_minions` fixture from conftest
       - Create states with known HP, minion counts, positions to verify potential
       - Verify shaped reward formula matches mathematical definition
       - Verify existing compute_reward is not broken

       - `CheckpointManager` class:
         - `__init__(self, pool_dir: Path, pool_size: int = 10)`
         - `save(self, model, step: int) -> Path` -- save model to `pool_dir/checkpoint_{step}.zip`, return path
         - `sample(self, latest_ratio: float = 0.5) -> Path | None` -- 50% chance latest, 50% random historical. Returns None if pool empty.
         - `list_checkpoints(self) -> list[Path]` -- sorted by step number ascending
         - `prune(self) -> None` -- keep only newest pool_size checkpoints, delete oldest
         - `pool_dir.mkdir(parents=True, exist_ok=True)` in __init__

    2. Create `src/grid_tactics/rl/self_play.py` with `SelfPlayEnv(gymnasium.Wrapper)`:
       - `__init__(self, env: GridTacticsEnv, opponent_policy=None, use_shaped_reward: bool = False, gamma: float = 0.99)`
         - `self.training_player_idx = 0` -- training agent is always player 0
         - `self.opponent_policy = opponent_policy` -- MaskablePPO instance or None for random
         - `self.use_shaped_reward = use_shaped_reward`
         - `self.gamma = gamma`
         - `self._prev_state = None` -- for potential-based shaping

       - `action_masks(self) -> np.ndarray` -- MUST delegate to `self.env.action_masks()` (per Pitfall 1)

       - `reset(self, **kwargs)`:
         - Call `self.env.reset(**kwargs)`, store initial state as `_prev_state`
         - If opponent moves first (`_is_opponent_turn()`), auto-step opponent
         - Return (obs, info) from training agent's perspective

       - `step(self, action)`:
         - Store `prev_state = self.env.state` before stepping
         - Call `self.env.step(action)` for training agent's action
         - If `use_shaped_reward` and not terminal: compute `compute_shaped_reward(prev_state, self.env.state, self.training_player_idx, self.gamma)` as reward
         - If not terminal/truncated, auto-step opponent turns in a while loop
         - After opponent auto-steps, re-compute reward from training agent's perspective if game ended during opponent turn
         - Update `_prev_state` for next call
         - Return (obs, reward, terminated, truncated, info)

       - `_is_opponent_turn(self) -> bool`:
         - Return `self.env._current_player_idx() != self.training_player_idx`

       - `_opponent_act(self) -> int`:
         - Get mask from `self.env.action_masks()`
         - If `self.opponent_policy is None`: sample uniformly from legal actions (`np.where(mask)[0]`)
         - Else: use `self.opponent_policy.predict(obs, action_masks=mask, deterministic=False)`
         - Return int action

       - `set_opponent(self, policy)` -- method to swap opponent policy mid-training

    3. Create `src/grid_tactics/rl/callbacks.py` with `SelfPlayCallback(BaseCallback)`:
       - `__init__(self, checkpoint_manager: CheckpointManager, save_freq: int = 10_000, verbose: int = 0)`
       - `_on_step(self) -> bool`:
         - If `self.n_calls % self.save_freq == 0`:

         - Return True
