    Create `src/grid_tactics/rl/training.py` with three public functions:

    1. **`create_model(env, seed=42, tensorboard_log="data/tb_logs", **kwargs) -> MaskablePPO`**
       - Returns `MaskablePPO("MlpPolicy", env, ...)` with these defaults (per research):
         - learning_rate=3e-4, n_steps=2048, batch_size=64, n_epochs=10
         - gamma=0.99, gae_lambda=0.95, clip_range=0.2
         - ent_coef=0.01 (exploration), vf_coef=0.5, max_grad_norm=0.5
         - verbose=1, seed=seed, tensorboard_log=tensorboard_log
       - `**kwargs` allows overriding any parameter
       - Import: `from sb3_contrib import MaskablePPO`

    2. **`evaluate_vs_random(model, env_or_factory, n_games=100, seed_start=10000) -> float`**
       - Plays n_games where the model plays as player 0 and random agent plays as player 1
       - Use SelfPlayEnv with opponent_policy=None (random) for evaluation
       - For each game:
         - Reset with incrementing seed (seed_start + i)
         - Loop: model.predict(obs, action_masks=mask, deterministic=True) -> step
         - Count wins (reward > 0 at termination)
       - Return win_rate = wins / n_games
       - Must handle both terminated and truncated states

    3. **`train_self_play(total_timesteps=100_000, db_path=Path("data/training.db"), checkpoint_dir=Path("data/checkpoints"), tensorboard_log="data/tb_logs", use_shaped_reward=True, save_freq=10_000, eval_freq=20_000, eval_games=50, description="", seed=42) -> dict`**
       - **Setup:**
         - Load CardLibrary from `data/cards`
         - Build standard deck (use all 18 starter cards with 3 copies each, truncating/padding to meet MIN_DECK_SIZE=40 -- or build a balanced 40-card deck from the starter pool)
         - Create GridTacticsEnv, wrap in SelfPlayEnv (with use_shaped_reward)
         - Create MaskablePPO via create_model()
         - Create CheckpointManager (pool_dir=checkpoint_dir, pool_size=10)
         - Create SelfPlayCallback (checkpoint_manager, save_freq)
         - Generate run_id: `f"run_{datetime.now(timezone.utc).strftime('%Y%m%d_%H%M%S')}"`
       - **Persistence (per D-07, D-08):**
         - Create TrainingRunWriter and GameResultWriter (db_path)
         - Call start_run with hyperparameters dict
         - Add a custom `GameLoggingCallback(BaseCallback)` that:
           - Counts completed episodes (on `_on_step`, check `self.locals["dones"]`)
           - After each episode: record_game() via GameResultWriter
           - Every eval_freq steps: evaluate_vs_random(), record_snapshot()
       - **Training:**
         - `model.learn(total_timesteps=total_timesteps, callback=[self_play_callback, game_logging_callback])`
       - **Cleanup:**
         - Save final model
         - Call end_run with total_timesteps and model_path
         - Flush writer buffers
       - **Return:** dict with run_id, final_win_rate (vs random), model_path, db_path

    4. Update `src/grid_tactics/rl/__init__.py` to export create_model, train_self_play, evaluate_vs_random.
