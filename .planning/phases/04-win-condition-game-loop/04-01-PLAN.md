    **Step 4: Add _apply_sacrifice to action_resolver.py** (per D-01, D-02, D-03)
    Create `_apply_sacrifice(state: GameState, action: Action, library: CardLibrary) -> GameState`:
    1. Find minion by action.minion_id, validate it belongs to active player
    2. Validate minion is on opponent's back row: P1 minion must be at row 4 (BACK_ROW_P2), P2 minion must be at row 0 (BACK_ROW_P1)
    3. Look up card definition from library to get base attack
    4. Calculate effective attack: `card_def.attack + minion.attack_bonus`
    5. Remove minion from board via `board.remove(row, col)`
    6. Remove minion from state.minions tuple
    7. Add minion's card_numeric_id to owner's graveyard
    8. Deal effective attack as damage to OPPONENT player via `player.take_damage(effective_attack)`
    9. Return updated state

    **Step 5: Add _check_game_over to action_resolver.py** (per D-05, D-06, D-07)
    Create `_check_game_over(state: GameState) -> GameState`:
    1. Check both players' `is_alive` property
    2. If both dead: `is_game_over=True, winner=None` (draw per D-06)
    3. If only P1 dead: `is_game_over=True, winner=PlayerSide.PLAYER_2`
    4. If only P2 dead: `is_game_over=True, winner=PlayerSide.PLAYER_1`
    5. If neither dead: return state unchanged
    Return updated state via replace().
