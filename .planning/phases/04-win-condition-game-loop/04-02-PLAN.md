    **Step 2: Create src/grid_tactics/game_loop.py** (per D-08, D-09, D-10, D-11)

    Define `GameResult` as a frozen dataclass:
    ```python
    @dataclass(frozen=True, slots=True)
    class GameResult:
        winner: Optional[PlayerSide]  # None = draw
        turn_count: int
        final_hp: tuple[int, int]     # (p1_hp, p2_hp)
        is_draw: bool
        reason: str                   # "hp_zero", "turn_limit", "deck_out" (discretion)
    ```

    Define `run_game(seed: int, deck_p1: tuple[int, ...], deck_p2: tuple[int, ...], library: CardLibrary, turn_limit: int = DEFAULT_TURN_LIMIT) -> GameResult`:
    1. Call `GameState.new_game(seed, deck_p1, deck_p2)` to get `(state, rng)`
    2. Main loop: while not `state.is_game_over` and `state.turn_number <= turn_limit`:
       a. Get `actions = legal_actions(state, library)`
       b. Use `rng.choice(actions)` to select uniformly random action (per D-09)
       c. Call `state = resolve_action(state, action, library)`
    3. After loop: build GameResult:
       - If `state.is_game_over`: winner from state.winner, reason="hp_zero"
       - If turn_number > turn_limit: winner=None, is_draw=True, reason="turn_limit"
       - final_hp = (state.players[0].hp, state.players[1].hp)
       - turn_count = state.turn_number
    4. Return GameResult

    Note on RNG: GameRNG needs a `choice` method. Check if it exists; if not, add a simple `choice(seq)` method to GameRNG that picks a random element from a sequence using the existing `self._rng` (numpy Generator). Use `self._rng.integers(0, len(seq))` as the index. This keeps the random agent deterministic with the game seed.

    Note on deck-out (Claude's discretion): If a player tries to draw from an empty deck and that's the only non-PASS action, the game naturally continues (they'll PASS or do other things). If both players can only PASS repeatedly and game stalls, the turn limit catches it. No special deck-out rule needed beyond the turn limit safety net.

    **Step 3: Extend GameRNG if needed**
    If `GameRNG` lacks a `choice()` method, add to `src/grid_tactics/rng.py`:
    ```python
    def choice(self, seq):
        """Select a random element from a non-empty sequence."""
        idx = int(self._rng.integers(0, len(seq)))
        return seq[idx]
    ```
    This maintains determinism through the seeded numpy Generator.
