   - For those games, compute matches against the caster's deck using the precomputed `tutor_target_table`:
     - Specific card_id: match where deck_card_id == target_card_id.
     - Dict selector: match where all provided attribute masks are satisfied.
   - Collect up to K=8 deck indices per game (pad with -1). Assert no game has > 8 matches (raise if so — current deck size prevents it).
   - Write into `pending_tutor_player[g] = caster_idx` and `pending_tutor_matches[g] = [idx0, idx1, ..., -1, -1]`.

   - Games that set pending MUST NOT trigger the react window this step.

2. Add `apply_tutor_select(state, games_mask, match_index_tensor)`:
   - For each active game: `deck_idx = state.pending_tutor_matches[g, match_index]`. Pop that deck index (shift deck tensor, decrement deck size), append to caster's hand tensor.
   - Clear `pending_tutor_player[g] = -1` and `pending_tutor_matches[g] = -1`.
   - Trigger react window for those games.

3. Add `apply_decline_tutor(state, games_mask)`:
   - Clear pending tensors. Deck/hand untouched. Trigger react window.
