# Frontend

Vanilla JS + Socket.IO client. Single-page web app served by [[Server|Flask]].

## Files
- `src/grid_tactics/server/static/index.html`
- `src/grid_tactics/server/static/game.js`
- CSS

## Features
- Lobby + room codes
- Deck builder
- 5x5 CSS Grid board with click targeting
- Hand display + dim-when-unaffordable
- Mana / HP / phase indicators
- [[../Mechanics/React Window]] modal
- [[../Mechanics/Tutor]] selection modal
- Game over overlay
- Game log sidebar
- [[Animation Queue]] for visual effects
