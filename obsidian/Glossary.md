## Trigger Keywords
| Keyword | Description |
|---------|-------------|
| Summon | Activates when minion is played. |
| Death | Activates when minion is destroyed. |
| Move | Activates when minion moves forward. See [[Movement]]. |
| Attack | Activates when minion attacks. See [[Range and Attacks]]. |
| Damaged | Activates when minion takes damage. |
| Passive | Triggers automatically every turn. |
| Active | Once-per-turn ability used instead of attacking. See [[Activated Abilities]]. |
